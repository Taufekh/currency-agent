# 💱 Currency Conversion Agent

An agentic AI currency converter built with LangGraph + Groq (free) + live exchange rates.

## Setup

1. Clone the repo
```bash
   git clone https://github.com/Taufekh/currency-agent.git
   cd currency-agent
```

2. Install dependencies
```bash
   pip install -r requirements.txt
```

3. Add your free Groq API key (get one at https://console.groq.com)
```bash
   cp .env.example .env
   # Edit .env and paste your key
```

4. Run
```bash
   python currency_agent.py
```
